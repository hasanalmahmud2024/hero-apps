# Hero IO - App Details

This repository contains the source code for the Hero IO application, a project designed to showcase app details and management functionality.

## Description

Hero IO is a web application built to display app information, including details, reviews, and installation status. It utilizes a JSON data structure to manage app data and provides features such as search, filtering, and local storage for installation tracking.

## Technologies Used

*   **React:**  For building the user interface.
*   **React-Router** For SPA.
*   **Tailwind** For faster styling.
*   **DaisyUI** For pre-styled components.
*   **JSON:**  For storing and managing app data.
*   **LocalStorage:** For persisting installation status.
*   **Recharts** For dynamic graphs.
*   **React-Toastify** For displaying toast notifications.
*   **Image DB** For hosting image.

## Features

*   **App Listing:** Displays a list of apps with key details (title, image, download count, rating).
*   **Search Functionality:** Allows users to search for apps by title.
*   **App Details Page:** Provides detailed information about a selected app, including reviews and a chart visualization.
*   **Installation Tracking:**  Allows users to "install" apps, with the status saved in local storage.
*   **Installation Page:** Displays a list of apps currently marked as installed.
*   **Responsive Design:** The application is designed to be responsive across different devices.
*   **Error Handling:** Includes custom error pages for invalid routes and search results.
*   **Sort by downloads** Allows users to sort by downloads high to low and vice versa.
